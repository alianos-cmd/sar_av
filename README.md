# Project Dirtle

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/alianos-cmd/sar_av
cd sar_av
pip install -r requirements.txt
```
git add README.md .gitignore
### Usage

```bash
python main.py
```
